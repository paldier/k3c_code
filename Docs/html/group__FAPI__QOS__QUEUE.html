Provides a set of API to create queue for an interface, configure shaper on a queue/port, etc.

Get Port QoS characteristics like rate shaper, and/or weight/priority if this port is cascaded into another scheduler.

Parameters
[in]ifnameName of interface whose port QoS characteristics are to be configured
[out]shaperPort shaper configuration
[out]weightPort weight when port is cascaded into next stage scheduler. -1 means no weight, only priority
[out]priorityPort priority when port is cascaded into next stage scheduler. -1 means no priority, only weight
[in]flagsFlags are defined as follows:
  • QOS_OP_F_INGRESS : if port egress or ingress configuration is to be returned
Returns
0 on successful / -1 on failure

char *  queue_name,
int32_t *  num_queues,
qos_queue_cfg_t **  q,
uint32_t  flags 
)

Get one of all QoS Queue Configuration on given interface.

Parameters
[in]ifnameName of interface on which queue is to be queried (egress or ingress)
[in]queue_nameName of queue which is to be queried
[out]num_queuesNumber of queues returned
[out]qArray of Queue configuration of num_queues number of queues Caller must free the buffer
[in]flagsSpecifies if the Filter is to do one of the following:
  • QOS_OP_F_INGRESS - if ingress queues are to be retrieved, otherwise egress queues are retrieved
Returns
0 on successful / -1 on failure
Remarks
if QueueId is -1, then all queues for if_name are returned (number of queues given by num_queues), else only queue given by queue_id is returned

Get statistics of one or all Queues on given interface.

Parameters
[in]ifnameName of interface on which queue is to be queried (egress or ingress)
[in]queue_nameName of queue for which statistics are to be queried
[out]num_queuesNumber of queues returned
[out]qstatsArray of Queue statistics of num_queues number of queues Caller must free the buffer
[in]flagsSpecifies whether Filter is to do one of the following: QOS_OP_F_INGRESS - if ingress queues statistics are to be retrieved, otherwise egress queues statistics are retrieved
Returns
0 on successful / -1 on failure
Remarks
if queue_name is not specified or NULL, then statistics of all queues for if_name are returned (number of queues given by num_queues), else only statistics of queue given by queue_name is returned
